Toy-Chest-Theme
===============

**Toy Chest** is a color scheme that makes coding look as fun as it really is.

### Text Editors

* **Vim**
* **TextMate**
* **TextMate 2**
* **Sublime Text 1, 2 & 3** (through TextMate)
* **BBEdit**
* **TextWrangler**

### IDEs

* **Coda 2**

### Terminal Emulators

* **Terminal.app**
* **iTerm2**

### Markdown Editors

* **Mou**
