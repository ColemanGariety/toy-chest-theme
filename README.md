Toy-Chest-Theme
===============

**Toy Chest** is a "Flat UI"-inspired color scheme for programmers who like to play.

Currently supports the following software:

### Editors & IDEs

* **Vim**
* **Coda**
* **BBEdit**
* **TextMate**
* **Sublime Text**
* **TextWrangler**

### Terminal Emulators

* **Terminal.app**
* **iTerm2**
